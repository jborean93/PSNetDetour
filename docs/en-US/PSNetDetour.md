# PSNetDetour Module
## Description
Hook .NET functions in PowerShell.

## PSNetDetour Cmdlets
### [New-NetDetourHook](New-NetDetourHook.md)
Creates a hook on a .NET method to intercept calls and modify behavior.

### [Use-NetDetourContext](Use-NetDetourContext.md)
Executes a scriptblock with automatic management of NetDetour hooks.
